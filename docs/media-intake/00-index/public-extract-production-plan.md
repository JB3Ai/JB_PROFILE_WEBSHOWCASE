# Public Extract Production Plan

## Milestone 19D Scope

Milestone 19D defines what each shortlisted `PUBLIC-EXTRACT` asset should
become before any extraction, editing, publication, or app integration starts.

This is a planning document only. It does not approve any raw source asset for
publication.

## Production Rules

- Raw source files stay uncommitted.
- Extracts must be reviewed before publication.
- Public outputs must be edited, redacted, compressed, and renamed before use.
- Do not publish private, legal, family, financial, court, evidence, or case material.
- iMED material must remain historical.
- GTR³ material must remain aspirational and symbolic.
- Do not wire any output into the website until an explicit integration milestone.
- Do not add files under `public/` until a future publication pass approves them.
