20U approves safe public-facing copy for the first controlled Batch 01 product rollout before any visual integration milestone.

The goal is to prepare product names, categories, status badges, descriptions, CTA behaviour, disclaimer notes, and restricted wording for future app-card implementation without rendering new public cards or promoting unreviewed assets.

This milestone keeps the portfolio pipeline disciplined:

copy approval first, asset review second, visual optimisation third, public rendering later.
